    🎨 Look & Feel

    This Hyprland config prioritizes a clean, modern aesthetic with smooth animations and subtle transparency effects, perfectly matching the custom Waybar.

    General Layout

    • Layout: Dwindle (default Hyprland dynamic tiling)
    • Gaps (Inner): 4px
    • Gaps (Outer): 8px
    • Border Size: 2px

    Borders

    • Active Border: A 45-degree gradient between Cyan (#33ccff) and Green (#00ff99).
    • Inactive Border: A dark semi-transparent grey (#595959aa).

    Decoration & Blur

    • Rounding: 6px rounded corners on all windows.
    • Blur: Enabled with a size of 3 and 1 pass for a subtle frosted glass effect behind windows.
    • Opacity: Active and inactive windows are set to 1.0 (fully opaque) for maximum readability.
    • Wallpaper: Managed by swaybg, set to scale a custom BlueCity wallpaper.

    Animations

    Animations are enabled using a custom bezier curve for smooth transitions: - Windows: 7ms smooth pop-in effect (80%). - Borders: 10ms color and angle transitions. - Fade: 7ms fade in/out. - Workspaces: 6ms transition when switching.

    Fonts

    • System/Bar Font: Z003 (install via AUR gsfonts)
    • Fallback Icons: JetBrainsMono Nerd Font
